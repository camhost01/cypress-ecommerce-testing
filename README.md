# 🧪 Cypress Automation - Laboratorio de Testing

This repository contains a basic E2E test automation solution using Cypress for the [https://www.laboratoriodetesting.com](https://www.laboratoriodetesting.com) site, as part of a QA strategy exercise.

---

## 📌 Covered functionalities

- User registration
- Add products to cart
- Checkout

---

## ⚙️ Requierements

- Node.js (v14 o superior)
- NPM

---

## 🚀 Installation

```bash
git clone https://github.com/TU_USUARIO/cypress-ecommerce-testing.git
cd cypress-ecommerce-testing
npm install
